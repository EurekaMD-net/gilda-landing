# gilda.mx — Landing Page

Landing page comercial para **Gilda**, el asistente de WhatsApp para salones de belleza en México.

## Stack
- HTML5 + CSS3 (vanilla, sin dependencias)
- JavaScript mínimo para interactividad
- Deploy target: Vercel / Netlify / gilda.mx

## Paleta de marca
| Color | Hex | Uso |
|-------|-----|-----|
| Coral | `#E8736A` | Primario, CTA |
| Teal | `#2ABFBF` | Secundario, acentos |
| Blanco | `#FFFFFF` | Fondo |
| Gris oscuro | `#2D2D2D` | Texto body |

## Secciones
1. Hero — propuesta de valor + CTA "14 días gratis"
2. El dolor — 3 problemas que resuelve Gilda
3. Cómo funciona — 3 pasos
4. Beneficios — agenda, recordatorios, reactivación
5. Precio — $599/mes, sin contrato
6. Social proof — placeholder para piloto
7. Footer — CTA final + contacto WA
